# P3-MS-02 - Milestone Derived Dates

## Scope

Verify that a Milestone linked to Releases derives its date range from the earliest linked Release start and latest linked Release end.

## Recorded DevInt execution

- Linked `RE-1` to `MS-2`.
- `RE-1` dates were Start `2026-07-01` and Release `2026-07-31`.
- Milestone Target Start derived to `2026-07-01`.
- Milestone Target End derived to `2026-07-31`.
- Both fields became read-only and displayed `Derived from linked Releases`.
- Removing the Release restored manual editing and retained the last values.
- Derivation recalculated on Save rather than in the unsaved preview.

## Result

Result: **Pass / Match**.

No product gap was confirmed. Save-time recomputation is accepted for the current scope.

## Test-data and evidence state

The Release link was removed after testing. No durable screenshot was committed; capture the linked and unlinked states during regression.
