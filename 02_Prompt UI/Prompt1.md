# Prompt 1 — Global Design System

Create a professional enterprise SaaS design system for an Agile Work Management Tool inspired by Rally Software layout.

The design must feel corporate, structured, dense, and productivity-focused.

Visual style:

* Clean enterprise software interface
* Light gray page background
* White panels and table areas
* Muted navy or blue as the primary color
* Minimal use of color
* Subtle borders
* Compact spacing
* Highly readable tables
* Professional business look
* No playful illustration
* No heavy gradient
* No colorful startup style

Typography:

* Use clean sans-serif font
* Clear hierarchy between page title, section title, table header, body text, metadata, and helper text
* Compact but readable table typography

Color system:

* Primary: muted blue or navy
* Background: light gray / off-white
* Surface: white
* Border: soft gray
* Text primary: dark gray
* Text secondary: medium gray
* Success: muted green
* Warning: muted amber
* Error/Critical: muted red
* Info: muted blue

Components to create:

* Top navigation bar
* Context selector bar
* Horizontal status summary strip
* Action toolbar
* List / Board toggle
* Table
* Table row
* Checkbox
* Icon button
* Primary button
* Secondary button
* Dropdown
* Search input
* Status badge
* Type badge
* Priority badge
* Avatar
* Progress bar
* Work item card
* Quick create row
* Empty state
* Modal
* Right-side detail panel

UX principle:
The product should prioritize clarity, speed, scanning, and structured enterprise workflow management.
